# CTF Development Repository

This repository contains materials and writeups for CTF challenges I've created and publicly released. This repository currently includes challenges from:
- USCGA AIM CTF 2022
- Nahamcon CTF 2024

## Organization

The challenges are organized by category and difficulty. Each challenge is marked with the CTF it was used in and its intended difficult.
- Web
- Cryptography
- Forensics
- Reverse_engineering
- Scripting
- Misc
